# KRAx

Sistema de gestão de empréstimos desenvolvido para facilitar o controle financeiro de clientes, parcelas, pagamentos e acompanhamento de cobranças.

## Sobre o projeto

O **KRAx** é uma aplicação web criada com foco na organização e gestão de empréstimos, permitindo o cadastro de clientes, controle de valores emprestados, acompanhamento de parcelas e registro de pagamentos de forma prática e estruturada.

O projeto foi pensado para atender cenários onde é necessário ter mais controle sobre informações financeiras, vencimentos, histórico de pagamentos e situação de cada cliente.

## Objetivos

- Centralizar o controle de empréstimos em um único sistema
- Organizar clientes, parcelas e pagamentos
- Facilitar o acompanhamento de vencimentos
- Melhorar a visualização da situação financeira de cada empréstimo
- Tornar a gestão mais prática, segura e eficiente

## Funcionalidades

- Cadastro de clientes
- Cadastro de empréstimos
- Controle de parcelas
- Registro de pagamentos
- Acompanhamento de vencimentos
- Visualização de histórico financeiro
- Organização de dados por cliente
- Interface voltada para produtividade e gestão

## Tecnologias utilizadas

- PHP
- MySQL
- JavaScript
- HTML5
- CSS3

## Estrutura do projeto

O sistema foi organizado para separar responsabilidades entre interface, regras de negócio e persistência de dados, facilitando manutenção e evolução do código.
